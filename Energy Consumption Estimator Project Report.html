1. Project Overview

The **BillBuddy: Energy Consumption Estimator** is a full-stack web application designed to help administrators manage and track customer energy usage, calculate associated costs in Indian Rupees (₹), and provide insightful reports and energy-saving tips. The system supports separate login experiences for administrators and registered customers.

Project Goal

To provide a robust, data-driven tool for estimating and managing household energy consumption and costs.

Monetary Basis

All cost calculations are based on the Indian Rupee (INR), with a hardcoded rate of ₹8.00 per Kilowatt-hour (kWh).

Target Users

Admin

Manages all customer data, application usage entries, and generates official reports.

Customer

Logs in to view their personal consumption data and cost analysis.

2. System Architecture

Three-Tier Architecture

The application uses a standard three-tier architecture, separating the presentation, business logic, and data layers for modularity and scalability.

1. Frontend (Presentation)

A user-facing Single-Page Application (SPA) built with HTML, CSS, and Vanilla JavaScript. Manages the UI and user interactions.

  • Files: `app.html`, `app.css`, `app.js`
  • Libraries: Tailwind CSS, Chart.js, jsPDF

2. Backend (Logic)

A RESTful API built with Python and Flask. Handles all business logic, calculations, authentication, and database communication.

  • File: `energy_app.py`
  • Actions: CRUD, Auth, Cost Calculation

3. Database (Data)

A local SQLite3 database that provides persistent storage for all application data, including users, appliances, and usage logs.

  • File: `app_database.py`
  • Tool: Python `sqlite3` library

3. Core Features

User Authentication & Roles

Secure login for two distinct user types:

  • Admin (`Admin` / `admin17193`): Full control over all data.
  • Customer (Email / Password): View-only access to their own data.

Master Data Management

Centralized control over key application data:

  • Application Catalog: A non-editable list of 50 household applications and their wattage.
  • Customer Management: Admin can Create, Read, Update, and Delete customer profiles.

Usage Tracking & Calculation

Users log daily usage (App, QTY, Hours) and the system automatically calculates:

Daily kWh: (Watts $\times$ Hours $\times$ QTY) $/$ $1000$
Daily Cost (₹): Daily kWh $\times$ ₹8.00

Reporting & Analytics

A powerful reporting view with dynamic filters:

  • Filter by Period: Daily, Monthly, Yearly, or Custom Date Range.
  • Application Chart: Breakdown of consumption by appliance.
  • Time Series Chart: Cost trends over the selected period.
  • Energy Saving Tips: Contextual tips based on usage.

PDF Report Generation

Administrators can select any customer and generate a downloadable PDF report. This report contains a detailed, itemized table of that customer's usage data, suitable for billing or records.
